# CLIProxyNew

Альтернативная реализация сервиса, оборачивающего CLI-агентов (Codex, Claude Code, Gemini CLI и др.) в OpenAI/Gemini/Claude-совместимый API.

Репозиторий вдохновлён [router-for-me/CLIProxyAPI](https://github.com/router-for-me/CLIProxyAPI), но проектируется как самостоятельный проект с другой архитектурой.

## Подход к upstream

Взаимодействие с upstream-провайдерами предполагается через **SDK upstream relay**, подключаемый как обычный Go-модуль (dependency в `go.mod`). Это позволяет развивать транспорт/реляцию отдельно от бизнес-логики сервиса и переиспользовать его в других проектах.

## Структура

```
cmd/         — точки входа (бинарники)
internal/    — внутренняя бизнес-логика (неэкспортируемая)
sdk/         — публичный SDK (контракты, типы, клиент)
```
