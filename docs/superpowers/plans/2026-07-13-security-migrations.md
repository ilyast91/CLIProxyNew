# Ф1: базовые миграции и security

**Цель:** заложить схему Postgres и безопасные примитивы для хранения API-ключей и обратимо шифруемых учётных данных.

1. Добавить падающие unit-тесты для bcrypt, AES-256-GCM/keyring и статической структуры миграций.
2. Реализовать bcrypt cost 12, разбор 32-байтного base64 master key и versioned keyring с AES-256-GCM.
3. Добавить начальную reversible-миграцию со всеми таблицами Ф1, дневной партицией `usage_events` и материализованным агрегатом.
4. Запустить форматирование, `go mod tidy`, целевые и полные тесты, `go vet ./...`.
